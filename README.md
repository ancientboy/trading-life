## 🔄 工作流程

### 信息流
1. **Intel Agent** 每5分钟扫描所有数据源
2. 过滤噪音，按 P0-P3 优先级排序
3. P0/P1 情报立即推送到 Redis Stream

### 分析流
4. **Analyst Agent** 接收情报，进行多维度分析
5. 生成结构化交易信号（带置信度）
6. 置信度 ≥ 60 的信号进入风控审核

### 执行流
7. **Risk Agent** 审核仓位、检查风控规则
8. 通过审核的信号等待主人确认（Telegram）
9. **Trader Agent** 执行交易（当前为模拟盘）

### 复盘流
10. **Reviewer Agent** 每日20:00自动复盘
11. 生成日报通过 Telegram 推送
12. 更新信号权重和策略参数

## ⚠️ 安全隔离

| 项目 | 说明 |
|------|------|
| 代码隔离 | `/opt/trading-agent/` 独立目录 |
| 数据隔离 | 独立数据库，不访问 daoltd 的 DB |
| API 隔离 | 使用模拟盘 API，不动真实资金 |
| 通知隔离 | 复用 Telegram，但标注 [模拟盘] |
